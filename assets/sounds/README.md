# 알림 사운드 파일

이 폴더에 다음 사운드 파일들을 추가해주세요:

- `alarm_01.wav` - 차임 (기본)
- `alarm_02.wav` - 딩동
- `alarm_03.wav` - 비프
- `alarm_04.wav` - 종소리
- `alarm_05.wav` - 트럼펫
- `alarm_06.wav` - 마림바
- `alarm_07.wav` - 기타

## 파일 형식 요구사항
- **포맷**: WAV 또는 MP3
- **길이**: 2-5초 권장
- **샘플레이트**: 44.1kHz 권장
- **Android**: WAV, MP3 모두 지원
- **iOS**: WAV 권장 (16-bit PCM)

## iOS 사운드 파일 복사
iOS에서 커스텀 알림 사운드를 사용하려면:
```bash
cp assets/sounds/*.wav ios/Runner/
```
